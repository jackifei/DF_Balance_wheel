# DF_Balance_wheel
两轮平衡小车
编程环境：vscode 插件：platformIO

硬件配置如下：
主控芯片：ESP32-S3-DevKitC-N16R8 因为要测试其他项目，所以买的N16R8，也可以买别的ESP32
陀螺仪：MPU6050
12V转3.3降压模块
SimpleFOC两路驱动模块
12V锂电池
2804直流无刷电机+AS5600磁编码器
